	Basically in this python script, I simply monitor the battery percentage and also look for sockets is get a plug or not based on that you will get notified which will warn you or about your battery percentage as well as on your screen notification will be appear in center of screen so that will draw your attention. 
	In order to use this script you simply need to copy BatteryStatusNotifier.pyw file and save it as whatever name you want. Then just copy or create shortcut of that file and paste it into the startup folder so whenever your computer is restarted this program will run in background and base on your battery status it will warn you.

Steps to do:
Step 1: Copy BatteryStatusNotifier.pyw or create shortcut of this file.
Step 2: Go to startup folder of windows. For go to startup folder do this:
		Step I: open RUN.
		Step II: type: shell:startup
Step 3: Paste BatteryStatusNotifier.pyw or shortcut file.
